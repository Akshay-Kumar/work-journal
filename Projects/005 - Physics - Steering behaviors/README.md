## Description
Demonstrating different physics behaviors in p5js:
- Seek
- Flee
- Align
- Separate

Inspired by Daniel Shiffman's Youtube Videos on [Autonomous Agents](https://www.youtube.com/watch?v=JIz2L4tn5kM&list=PLRqwX-V7Uu6YHt0dtyf4uiw8tKOxQLvlW).
